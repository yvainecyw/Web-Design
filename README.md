# 原住民推廣平台
- 這是一個課堂分組專題，主要是設計一個布農族地區的旅遊網站來幫助偏向地區，我負責的部分是訂房資訊。
- 使用 Node.js、Boostrap4、HTML5、CSS3、jQuery 進行編寫。
